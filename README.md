# makeAppIcons

To generate iOS App Icons from command line.

## Usage

```
$ cd makeAppIcon
$ bundle exec bin/makeAppIcons --in sample/AppIcon.png  --out ./out
````

## Expects
- Platform has `sips` command.
- Input file must be PNG format.
- Input file should be larger than 1024x1024.
